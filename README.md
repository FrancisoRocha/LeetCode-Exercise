# Ejercicios LeetCode

[![JavaScript](https://img.shields.io/badge/JavaScript-ES6-yellow?logo=javascript&logoColor=black)](https://www.javascript.com/) [![Node.js](https://img.shields.io/badge/Node.js-%3E%3D14-43853d?logo=node.js&logoColor=white)](https://nodejs.org/) [![License: MIT](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

> Repositorio personal para almacenar soluciones a ejercicios de LeetCode.

Descripción
---
Este repositorio contiene soluciones a problemas de LeetCode que iré resolviendo, con implementaciones en JavaScript, notas sobre complejidad y ejemplos de uso.

Estructura
---
- Cada problema vive en su propia carpeta con el prefijo numérico: `01-Problem-Name/`.
- Dentro de cada carpeta: el archivo de solución (por ejemplo `romanToInteger.js`) y un `README.md` opcional con explicación.

Ejemplo
---
- Carpeta: `01-Roman-To-Integer/`
- Archivo: `01-Roman-To-Integer/romanToInteger.js`

Cómo ejecutar
---
Requiere Node.js (recomendado >= 14).

Ejecutar un archivo de solución:

```
node 01-Roman-To-Integer/romanToInteger.js
```

Convenciones
---
- Lenguaje principal: JavaScript.
- Nombres de carpeta: `NN-Problem-Name` (NN = índice con dos dígitos).
- Mensajes de commit descriptivos: `feat: add solution for Roman To Integer`.

Contribuir
---
- Si quieres proponer una solución o mejora, crea un fork y envía un pull request.
- Añade tests o ejemplos si es posible.
